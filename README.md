# ML Dataset Profiler

A lightweight utility to automatically analyze datasets (CSV) and generate a summary report including:
- Missing values
- Data types
- Numeric stats (min/max/mean/std)
- Duplicate detection
- Column suggestions for ML preprocessing

## Features
- Simple CLI usage
- Works with any CSV dataset
- Outputs clean text report

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python src/profiler.py your_dataset.csv
```
